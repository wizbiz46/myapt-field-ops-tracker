# Local edits and sheet update exports

V1 uses localStorage for in-app edits. Use exports to move changes back into source sheets.

## Add/edit partners

Use Partners → Add partner, or open a partner → Edit partner.

Export with:

- Partners → Export partners
- Settings → Export all CSVs

Import/update the exported partner CSV into the partner source sheet/data.

## Edit building info

Open Media → open a building → Edit building info.

Use this for missing/incorrect:

- media status
- neighborhood
- management company
- website
- FP visibility
- # floorplans
- Media Strategy 2 raw value
- notes
- DNP
- muted alerts

Export with:

- Media → Export building updates
- Settings → Export all CSVs

Use the exported CSV to update Alerts Sheet / Building Master.

## Captured units

Use Capture Log → Add capture, or open a building → Add captured unit.

Export captured unit logs via Settings → Export all CSVs.
